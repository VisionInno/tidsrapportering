# Tidsrapportering - Kom igång

## Starta appen

1. Öppna en terminal/kommandotolk i projektmappen
2. Skriv: `npm run dev`
3. Öppna webbläsaren och gå till: **http://localhost:5173**

---

## Använda appen

### Logga tid

1. Välj **datum** i kalenderfältet
2. Välj vilket **projekt** du jobbat med
3. Skriv en kort **beskrivning** av vad du gjorde
4. Fyll i antal **timmar** (t.ex. 2.5 för två och en halv timme)
5. Kryssa i "Fakturerbar" om tiden ska faktureras
6. Klicka på **Lägg till**

### Se din statistik

- **Överst** ser du totalt antal timmar, fakturerbara timmar och summa att fakturera
- **Stapeldiagrammet** visar hur mycket du jobbat senaste 7 dagarna
- **Cirkeldiagrammet** visar fördelningen mellan projekt

### Hantera projekt

1. Klicka på **Projekt** uppe till höger
2. Här kan du:
   - Lägga till nya projekt
   - Inaktivera projekt du inte längre använder
   - Ta bort projekt

### Exportera rapport

1. Klicka på **Exportera** uppe till höger
2. Välj format: **CSV** (för Excel) eller **PDF**
3. Välj tidsperiod: Senaste veckan, denna månad, eller eget datum
4. Klicka **Exportera** - filen laddas ner automatiskt

---

## Bra att veta

- All data sparas lokalt i din webbläsare
- Du behöver ingen inloggning eller internet
- Stäng inte terminalen medan du använder appen
- För att stänga appen: tryck `Ctrl + C` i terminalen
